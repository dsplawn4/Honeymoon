# The Grand Tour 2025

Interactive travel itinerary — 66 days, Lima to Lima.

## Deploy to Vercel (free, 5 minutes)

1. Create a free account at **github.com**
2. Create a new repository called `grand-tour`
3. Upload all these files to the repo
4. Go to **vercel.com**, sign in with GitHub
5. Click **Add New Project** → import your `grand-tour` repo
6. Vercel auto-detects Vite — just click **Deploy**
7. You get a live URL like `https://grand-tour.vercel.app`

## Local development

```bash
npm install
npm run dev
```
